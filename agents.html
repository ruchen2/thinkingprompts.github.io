Agents Method

  • Describe the problem and ideal solution. What is the problem and ideal solution?
  • Choose the starting point between the problem and ideal solution. What is important?
  • Create a graphical solution. It is possible to create from the problem, from the solution, or from both ends. What does a solution looks like?
  • Develop a structure logic for the agents' actions. What are the agents doing?
  • Prepare the logic list of agent properties in relation to actions that were previously decided. What properties must the agents have?
  • Generate solutions. How can agents be transformed into a technical system or technological process?
