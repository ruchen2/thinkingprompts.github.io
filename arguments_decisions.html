Analyze Arguments and Decisions

  • Argument is the process of giving a reason in support of a claim/conclusion
  • Augument = Reason + Claim/Conclusion
  • Words that signal reasons: since, given that, because, for the reason that, suppose, assume, let's begin agreeing that, the evidence is as follows, we all know that, in the first place, is supported by, is implied by, is derived from the fact that
  • Words that signal conclusions:so, thus, therefore, hence, we can now infer, it follows that, this means that, this implies that, these facts indicate, the evidence shows, let's infer that, so it would seem that, and so probably, we can deduce, this supports the view that
